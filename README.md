# VELOCRAFT: Engineered Beyond Speed

![VELOCRAFT Hero](public/assets/car-renders/hero-car.jpg)

**VELOCRAFT** is a premium, Awwwards-level concept website for an ultra-luxury electric hypercar brand. It features a cinematic, immersive user experience with high-end 3D-style styling, smooth scrolling physics, and a fully responsive layout.

## ✨ Features

- **Awwwards-Level Interactions:** Implements `@studio-freight/lenis` for buttery-smooth scrolling and custom magnetic cursor effects.
- **Cinematic Reveal:** Includes a dark, immersive preloader that builds anticipation before revealing the hypercar.
- **Mouse Parallax:** Features interactive 3D perspective tracking on the hero car image based on user mouse movement.
- **SEO & Performance Optimized:** Built with Vite, React, and Tailwind CSS. Fully equipped with Open Graph tags, Twitter Cards, Schema.org JSON-LD, and font preloading.
- **Progressive Web App (PWA):** Installable as a native-like app on desktop and mobile devices via `vite-plugin-pwa`.
- **Fully Responsive:** Gracefully scales from massive 4K desktop displays down to mobile devices.

## 🚀 Tech Stack

- **Framework:** [React 18](https://react.dev/) + [Vite 6](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://lenis.studiofreight.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/velocraft-hypercar-web.git
   cd velocraft-hypercar-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`.

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🌐 Deployment

This project is perfectly configured for seamless deployment on **Vercel**:

1. Push this repository to GitHub.
2. Go to your [Vercel Dashboard](https://vercel.com/dashboard) and click **Add New... > Project**.
3. Import your GitHub repository.
4. Leave the Framework Preset as **Vite**.
5. Click **Deploy**. Vercel will automatically build and publish the site.
